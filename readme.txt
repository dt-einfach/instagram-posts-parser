*. Заполни файл "config.py" в папке проекта

*. Запусти "s1_direct_messages.py". Он скачает сообщения из нужного чата ПМ

Рекомендации по выполнению скрапинга:
- страницу браузера под контролем скрипта не нужно сворачивать или менять в размере, т.к. скрипт работает с вёрсткой, а она, как водится, адаптивная;
- во время выполнения скрипта лучше не водить курсором по странице браузера, т.к. многие элементы на него реагируют.

*. Запусти "s2_save_posts.py". Во время первого запуска скрипт откроет страницу инстаграма и будет ждать, что ты самостоятельно залогинишься. Залогинься, затем в терминале нажми Enter. Скрипт сохранит куки для входа, поэтому больше логиниться будет не нужно. Далее скрипт пройдёт по всем постам из ПМ, сохранит их в Saved и подпишется на авторов.

*. Запусти "s3_subscribe_all.py". Когда скрипт откроет страницу инстаграма, нажми в терминале Enter (тем самым подтверди, что сайт открылся нормально). Скрипт пройдёт по всем ссылкам на инстаграм из ПМ и подпишется, если это ссылка на аккаунт.

*. Запусти "s4_download_saved.py". Когда скрипт откроет страницу инстаграма, нажми в терминале Enter (тем самым подтверди, что сайт открылся нормально). Скрипт откроет страницу Saved и скачает все медиа с постов оттуда в папку "downloads". На один пост нужно около 10-20 секунд, поэтому общее время выполнения может быть большим. Если скачивание постов по какой-то причине прервётся, то при повторном запуске скрипт пропустит ранее скачанные посты.
